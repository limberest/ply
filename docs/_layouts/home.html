Exercise
You're familiar with JavaScript, and you need automated tests to put your API through its paces.
  1. Export a Postman collection with the parameterized requests that'll form the basis of your test cases
    • Runtime values are supplied by standard JSON object files, or can be assigned through script
  2. Write a test case that orchestrates requests
    • Every run produces a highly-readable YAML result file
    • Expected results are compared versus actual to determine test case success
